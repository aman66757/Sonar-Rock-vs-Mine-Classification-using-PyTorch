# Sonar-Rock-vs-Mine-Classification-using-PyTorch
This project implements a Fully Connected Neural Network (FCNN) using PyTorch to classify sonar signals as Rock or Mine. It compares models with and without Dropout Regularization to understand its impact on reducing overfitting and improving generalization.

### Features
- Data preprocessing using Pandas and Scikit-learn
- Label Encoding
- StandardScaler
- Train-Test Split
- PyTorch TensorDataset & DataLoader
- Fully Connected Neural Network (FCNN)
- Adam Optimizer
- CrossEntropyLoss
- Dropout Regularization
- Model Evaluation
- Training & Validation Loss Visualization

### Technologies Used
- Python
- PyTorch
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
